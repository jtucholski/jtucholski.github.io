  • int x;

  • (isOn) ? "It is on" : "It is not on">

    If the isOn variable is true, use the value "It is on", otherwise use the value "It is not on".

  • for(int i = 0; i < 10; i++)>

    Run a for loop starting i at 0 and incrementing it by 1 each time. Quit when i is >= 10.

    Call the CalculateArea method from this class's superclass.

  • itemsInCart == 1>

    Compare the value from itemsInCart and give the value true if they are equal, false otherwise.

    Multiple the values of the variables height and width and store the value in the variable area.

  • person.GetFullName()>

    Call the GetFullName() method on the object stored in the person variable.

  • What is an interface?

    >

    An interface is a contract that specifies which methods a class must have. It does not specify how the class must implement the method.

  • x++;>

    Equilavent of incrementing the value of x by one.

  • What is a code block?

    >

    A block permits multiple statements of code to be grouped together.

  • if (x == 5 && y < 7) { } >

    Execute the code between the braces if x is equal to 5 AND y is less than 7.

  • name = names[0]; >

    Access the first value in the names array and store a reference to it in the name variable.

  • int[] numbers; >

    Declare a variable called numbers that can reference an array of integers.

  • numbers = new int[5]; >

    Initialize a new array of integers of length 5 and store the reference in the variable called numbers.

  • while(hasMoreRows) { } >

    Execute the code in the while block continuously while hasMoreRows is equal to true.

  • x = numbers[numbers.Length - 1]; >

    Get the value from the last index in the numbers array and store a copy in the variable x.
